1. About change log file
The files name contains the language code  because the user manual and changes log are also published on the RPTools.org website.

2. Attention after converting addonUserManual.t2t and changes_.t2t with txt2tags:

The converter produces  addonUserManual.xhtml and changes_lang.xhtml files.
You have to manually change the extensions xhtml to html by renaming the files.


3. Translation of the user manual
The user manual was originally written in French;
The English translation was made with Google translate and revised by the author. Using this translation as the basis for a new translation could lead to errors.

The Spanish translation was done by a human, Remy Ruiz;

The Portuguese translation appears to have been done by Google Translate and then lightly revised by Portuguese humans ;
